# #PROJECT_NAME#

基于 Ethan CLI 构建的 Vue 3 + Vite + Ant Design Vue 项目。

## 技术栈

- Vue 3
- Vite 5
- Vue Router 4
- Pinia 2
- Ant Design Vue 4
- Tailwind CSS 3
- ESLint 9 (Flat Config)
- Prettier

## 开发

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产环境

```bash
npm run build
```

### 代码检查

```bash
npm run lint
```
